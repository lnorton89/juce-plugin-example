## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Script output to console | Scripts must not echo secret values to stdout/stderr or log files |
| Cloudflare API token to script execution | Token is read from environment variable, never written to files |
| Script state to generated-state.json | Account IDs written to gitignored file only; validated before writing |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04-06 | Information Disclosure | infra/deploy.ps1 | mitigate | secret put reads from env vars, never from args or files |

| T-04-10 | Repudiation | infra/teardown.ps1 | mitigate | Print exact wrangler commands before confirmation |
