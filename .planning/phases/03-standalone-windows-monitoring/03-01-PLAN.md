<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Windows/JUCE input device -> native capture adapter | Untrusted device callback shape, channel count, block size, and samples enter native code. |
| native source state -> WebView protocol | Native source descriptors and errors cross into React as bounded protocol-v1 payloads. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01-01 | Tampering | `source.select`/source descriptors | mitigate | Use closed `SourceMode` enums, bounded IDs/names, and parser/builder tests; never infer mode from display text. |
| T-03-01-02 | Information Disclosure | source fallback behavior | mitigate | Enforce D-06: failed selected input stops and asks the user to choose again, preventing silent capture of another microphone. |
| T-03-01-03 | Denial of Service | JUCE input callback | mitigate | Preallocate conversion buffers outside callbacks and grep/test for JSON/WebView/file/network/blocking work in callback paths. |
| T-03-01-SC | Tampering | npm/pip/cargo installs | accept | No package-manager installs are planned; use pinned JUCE and existing Windows SDK/JUCE modules. |
</threat_model>
