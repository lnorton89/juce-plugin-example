## 7. Key Findings

1. **Product identity** (name, company, version) appears in 12+ locations across 4 languages — highest priority for centralization
2. **Grace period 7** is duplicated as both `GraceModel::gracePeriodDays` (7.0) and a raw literal `7` in `LicensingCore.cpp` — should be unified
3. **Warning threshold mismatch**: C++ uses `warningThresholdDays = 1.0` but UI checks `>= 3` days — these must be reconciled
4. **Test key ID** `"test-ed25519-2026-06"` contains a hardcoded year/month that will become stale
5. **Event/property name strings** have no shared schema between C++ and TypeScript — pure convention currently
6. **Filesystem paths** (`entitlement.dat`, `source-preference.json`, `LumaScope/` dir) are hardcoded in 2 different store implementations
