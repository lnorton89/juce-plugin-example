## Sampling Rate

- **After every task commit:** Run `node scripts/validate-config.mjs`
- **After every plan wave:** Run full suite (CMake configure + build + tests + UI tests)
- **Before `/gsd-verify-work`:** Full suite must be green
- **Max feedback latency:** 120 seconds
