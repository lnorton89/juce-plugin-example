## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| host audio callback -> analyzer ingress | Real-time host thread supplies untrusted block sizes/sample values under strict timing constraints. |
| audio thread -> message/UI thread | Complete snapshots cross from real-time producer to non-real-time consumer. |

| T-02-05 | Repudiation | passthrough behavior | mitigate | Add deterministic before/after buffer tests for mono/stereo layouts so transparent behavior is auditable. |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | No package-manager installs are planned. |
</threat_model>
